# TP2 - Organización Empresarial: Análisis de Ventas

## Descripción
Este proyecto analiza un conjunto de datos simulados de ventas comerciales para generar indicadores básicos de desempeño.

## Estructura
* `/datos`: Contiene el dataset `ventas.csv`.
* `/scripts`: Contiene la lógica en Python (`analisis_ventas.py`).
* `/resultados`: Almacena los gráficos generados.

## Ejecución
El código está diseñado para ejecutarse en entornos virtuales como Google Colab utilizando rutas relativas.
